🎮Starting a Game
1
The host opens the Host Game tab, enters their name, picks a question count, timer, and categories, then clicks Create Game.
2
A unique Game ID and QR code appear in the lobby. Share either with your friends.
3
Players open the Join Game tab, enter the Game ID and a display name, then click Join Game.
4
The lobby stays open for 25 seconds. After that it locks automatically and the game begins — no late joins allowed.
❓Answering Questions
1
Everyone receives the same question at the same time. Questions are either multiple choice (4 options) or True / False.
2
Tap your answer before the countdown timer reaches zero. Once submitted your answer is locked — no changes allowed.
3
After the timer ends, the correct answer is revealed and the leaderboard updates for 6 seconds before the next question.
🏆Scoring
Correct answer +1.0 pt
Speed bonus +0.05 × seconds left
Wrong or no answer 0 pts
Example: 35s timer — you answer correctly with 20s remaining
Score = 1.0 + (0.05 × 20) = 2.0 points
💡Tips
⚡Answer fast — the speed bonus can be worth more than the base point.
📱Scan the QR code in the lobby to share the game instantly — no typing needed.
🔄Accidentally refresh? A Rejoin prompt will appear to get you back into the game.
🛑The host can end the game at any time from the End Game button — final scores are shown immediately.
🏷️Equal scores share the same rank — ties are shown fairly on the leaderboard.
